# Dossier Public pour les Images et Assets de Zara Gallery

Déposez vos images dans ce dossier `/public`. Elles seront accessibles directement à la racine de l'application et par les navigateurs :

1. **`favicon.ico`** : Renommez votre icône de favori en `favicon.ico` et déposez-la ici afin qu'elle s'affiche sur les onglets du navigateur.
2. **`logo.png`** : Déposez votre logo sous le nom `logo.png` ici pour qu'il soit utilisé comme logo principal de l'application, icône d'accueil Apple et presets.
3. **Autres images** : Vous pouvez déposer n'importe quel fichier image ou icône ici (ex: `banniere.jpg`, `produit-modele.png`), puis y faire référence directement dans l'application avec le chemin `/nom-du-fichier.ext` (par exemple `/banniere.jpg`).

---

*Note : Les fichiers du dossier `/public` sont servis de manière statique et ne subissent aucune modification par le compilateur.*
